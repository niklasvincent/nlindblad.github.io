Demystifying HTTP security headers

This page provides a list of resources that are a good follow up to my talk Demystifying HTTP security headers.

1. Strict-Transport-Security

It might be worth knowing how to reset the Strict Transport Security settings for a particular domain in your browser. See here for Chrome. For Firefox it will depend on which version you are using.

Resources

  1. List of browsers supported
  2. Understanding HTTP Strict Transport Security (HSTS) and preloading it into the browser by Troy Hunt
  3. HTTP Strict Transport Security on Mozilla Developer Network
  4. HTTP Strict Transport Security on The Chromium Projects wiki

2. Content-Security-Policy

The developer tools in Chrome will give you a good overview of which resources that get blocked due to the Content-Security-Policy whilst you are developing.

Resources

  1. List of browsers supported
  2. Implementing Content Security Policy on the Mozilla HACKS blog
  3. Content-Security-Policy generator
  4. Another Content-Security-Policy generator
  5. HTTP Strict Transport Security on Mozilla Developer Network
